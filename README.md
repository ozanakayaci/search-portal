## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ozanakayaci/search-portal.git
   ```
2. Go to directory

   ```sh
   cd search-portal
   ```

3. Install NPM packages
   ```sh
   npm install
   ```
4. Runs the app in the development mode.

   ```sh
   npm start
   ```
